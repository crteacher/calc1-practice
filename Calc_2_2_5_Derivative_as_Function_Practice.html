The Derivative as a Function

Calculus I  ·  §2.2.5  ·  Instructor: Christopher Halverson  ·  Fall 2026

Core Ideas
The Derivative as a Function $$f'(x) = \lim_{h \to 0} \frac{f(x+h) - f(x)}{h}$$ Same definition as before, but $x$ is now a variable. The result is a new function whose output at any $x$ equals the slope of $f$ there.
Key Results & Reading f' $f(x)=c \Rightarrow f'(x)=0$    $f(x)=ax+b \Rightarrow f'(x)=a$    $f(x)=ax^2 \Rightarrow f'(x)=2ax$
If $f'(x)>0$: $f$ is increasing.   If $f'(x)<0$: $f$ is decreasing.   If $f'(x)=0$: horizontal tangent.
