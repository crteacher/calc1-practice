First Derivative Test

Calculus I  ·  §3.6  ·  Instructor: Christopher Halverson  ·  Fall 2026

Core Ideas
Increasing / Decreasing $\frac{d}{dx}[f]>0 \;\Rightarrow\; f$ increasing  •  $\frac{d}{dx}[f]<0 \;\Rightarrow\; f$ decreasing. Find critical numbers, then test the sign of $f'$ in each interval between them.
First Derivative Test At critical number $x=c$:   $f'$ changes $+\!\to\!-$ $\Rightarrow$ local max  •  $f'$ changes $-\!\to\!+$ $\Rightarrow$ local min  •  no sign change $\Rightarrow$ neither.
